SW_LP_x_CA_1B

LP calibration parameters

Thematic areas: Plasma measurements

Applicable missions: Swarm

Applicable spacecraft: Swarm-A, Swarm-B, Swarm-C

Description

The LP_X_CA_1B Product contains the TCF.LP Data Set records, that is a the Temporal Calibration File containing the LP calibration parameters determined in the L1B Processor from the Offset Determination Sweep Mode telemetry including the measurements themselves.

Each product comes in a zip file composed of one XML Header file, one Product file and one Quality Test Report file:

  • XML Header file with extension .HDR
  • Product File with extension .cdf
  • Quality Test Report file with extension .EEF
