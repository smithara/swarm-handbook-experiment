SW_VFMxAUX_1B

Spacecraft (S/C) magnetic stray fields and Sun induced stray field at VFM sensor position

Thematic areas: Magnetic measurements

Applicable missions: Swarm

Applicable spacecraft: Swarm-A, Swarm-B, Swarm-C

Description

The VFMxAUX_1B Product contains detailed information on the S/C magnetic stray fields and the Sun induced stray field at the VFM sensor position. The data are provided at 50 Hz at the time-instants of the VFM measurements; the stray field vectors are provided in the VFM sensor frame.

Each product comes in a zip file composed of one XML Header file, one Product file and one Quality Test Report file:

  • XML Header file with extension .HDR
  • Product file with extension .cdf
  • Quality Test Report file with extension .EEF
