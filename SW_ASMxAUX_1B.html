SW_ASMxAUX_1B

Spacecraft (S/C) magnetic stray fields at ASM sensor position

Thematic areas: Magnetic measurements

Applicable missions: Swarm

Applicable spacecraft: Swarm-A, Swarm-B, Swarm-C

Description

The ASMxAUX_1B Product contains detailed information on the S/C magnetic stray fields and the Sun induced stray field at the ASM sensor position. The data are provided at 50 Hz at the time-instants of the VFM measurements (for internal practical purposes); the stray field vectors are provided in the ASM sensor frame.

Each product comes in a zip file composed of one XML Header file, one Product file and one Quality Test Report file:

  • XML Header file with extension .HDR
  • Product file with extension .cdf
  • Quality Test Report file with extension .EEF
