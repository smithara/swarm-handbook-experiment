SW_MODx_SC_1B

Satellite Position and velocity from the preliminary Medium Accuracy Orbit Determination

Thematic areas: Ephemeris

Applicable missions: Swarm

Applicable spacecraft: Swarm-A, Swarm-B, Swarm-C

Description

Position and velocity from the preliminary Medium accuracy Orbit Determination (MOD) expressed with respect to the International Terrestrial Reference Frame (ITRF).

Each product comes in a zip file composed of one XML Header file, one Product file and one Quality Test Report file:

  • XML Header file with extension .HDR
  • Product file with extension .sp3
  • Quality Test Report file with extension .EEF

The .sp3 product store the medium precision orbit information in ASCII format, following the Extended Standard Product 3 Orbit Format (SP3c)
